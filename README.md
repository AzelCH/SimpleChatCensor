# SimpleChatCensor
A simple PocketMine-MP plugin who censor bad words

# Features
- [x] Add words
- [x] Custom replace format
- [ ] Bypass
- [ ] Setting in game

# Configuration
```---
#ChatCensor Configuration

#replace format
replace-format: "****"

#badwords
#you can add or remove the badwords
badwords:
  - "fuck"
  - "dick"
  - "pussy"
  - "tits"
  - "noob"
  - "shit"
  - "cum"
  
...
```
